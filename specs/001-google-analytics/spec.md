# Feature Specification: Google アナリティクス追加

**Feature Branch**: `001-google-analytics`
**Created**: 2026-03-20
**Status**: Draft
**Input**: User description: "Google アナリティクスを追加したい"

## User Scenarios & Testing *(mandatory)*

### User Story 1 - サイト訪問データの計測 (Priority: P1)

サービス運営者として、ShipInfo サイトへの訪問者数・ページビュー・流入経路などの基本指標を Google アナリティクスで確認できる。

**Why this priority**: サイトの利用状況を把握することが計測機能の最低限の価値であり、すべての分析はここから始まる。

**Independent Test**: Google アナリティクス管理画面にアクセスし、サイトへの訪問がリアルタイムで記録されることを確認することで独立してテスト可能。

**Acceptance Scenarios**:

1. **Given** サイトが公開されている、**When** ユーザーがトップページを開く、**Then** GA4 にページビューイベントが送信される
2. **Given** サイトが公開されている、**When** ユーザーが運航詳細ページに移動する、**Then** 遷移先の URL がページビューとして記録される
3. **Given** GA の計測が有効な環境、**When** 複数ページを回遊する、**Then** セッション・ユーザー数・流入元が管理画面に表示される

---

### User Story 2 - 開発環境での計測除外 (Priority: P2)

開発者として、ローカル開発環境での操作が本番の計測データに混入しない。

**Why this priority**: 開発・テストのアクセスが混入するとデータ品質が低下し、意思決定に悪影響を与える。

**Independent Test**: ローカル環境でサイトを開き、GA 管理画面のリアルタイムレポートに該当アクセスが現れないことを確認。

**Acceptance Scenarios**:

1. **Given** 開発環境（APP_ENV=dev）でサイトを開く、**When** ページを閲覧する、**Then** GA へのイベント送信が行われない
2. **Given** 本番環境（APP_ENV=prod）でサイトを開く、**When** ページを閲覧する、**Then** GA にイベントが送信される

---

### Edge Cases

- 広告ブロッカー等によって GA スクリプトの読み込みがブロックされた場合、サイトの表示・機能に影響が出ないこと
- 測定 ID が未設定の場合、エラーを出さずにスクリプトの挿入をスキップすること
- ユーザーが JavaScript を無効にしている場合、ページ表示には影響しないこと

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: システムは全ページのヘッダー部分に GA4 のトラッキングスクリプトを埋め込まなければならない
- **FR-002**: GA4 の測定 ID（G-XXXXXXXXXX 形式）は環境変数または設定ファイルで管理し、コードにハードコードしてはならない
- **FR-003**: 本番環境でのみトラッキングが有効になり、それ以外の環境では無効にしなければならない
- **FR-004**: 測定 ID が未設定の場合はトラッキングスクリプトを出力しないこと
- **FR-005**: トラッキングスクリプトの有無によって、既存のページ機能・デザインに影響を与えてはならない

### Key Entities

- **測定 ID**: GA4 プロパティを一意に識別する文字列（G- で始まる）。環境ごとに異なる値を設定できる

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 本番サイトの全ページで GA4 管理画面にページビューが記録される（計測漏れ 0 件）
- **SC-002**: 開発環境のアクセスが本番 GA データに混入しない（開発アクセスによるデータ汚染 0 件）
- **SC-003**: トラッキングスクリプトを追加しても、既存ページの表示崩れや機能不全が発生しない
- **SC-004**: 測定 ID の変更がコード修正なしに設定のみで完結できる

## Assumptions

- Google アナリティクス 4（GA4）を使用する（Universal Analytics は 2023 年にサポート終了済み）
- GA4 プロパティおよび測定 ID はプロジェクト管理者が別途作成・取得済みとする
- プライバシーポリシーへの GA 利用記載はこの機能のスコープ外とする
- Cookie 同意バナーの実装はこの機能のスコープ外とする（将来的に別フィーチャーで対応）
